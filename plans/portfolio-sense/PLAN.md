# 方向 7：持仓感知系统 — PLAN

> 前面 6 个方向都在分析市场。方向 7 分析**你和市场的关系**。

---

## 现状

你现在有三套各自完整但互相隔离的系统：

```
longbridge-terminal          xueqiu-analyzer               xueqiu-monitor
   │                              │                              │
   ├─ 你持有的真实仓位              ├─ 对单股生成deep分析           ├─ 64 只自选股的情感追踪
   ├─ 你的买入成本+盈亏             ├─ 8 维度评分                  └─ Z-score 变化告警
   └─ 你的交易历史                 └─ PDF 公告全文提取
```

**没有人回答这个问题**：综合这三条信息，你现在应该做什么？

---

## 核心能力

### 一个例子

```
系统发现:
  ├─ longbridge: 你的茅台持仓成本 ¥1,200，当前 ¥1,326，浮盈 10.5%
  ├─ xueqiu-monitor: 茅台情感连续 3 周走低 (0.05→-0.08→-0.15)
  └─ xueqiu-analyzer: 上次评分 155，公告中分红方案已发布
      │
      ▼
  "你的茅台浮盈 10.5%，但雪球情感在持续走低。
  情感连续走低 3 周以上的情况，历史上发生过 4 次，
  后续 30 天均价涨跌幅: +3%, -1%, -5%, -2%。
  中位数 -1.5%。
  可以设置一个止盈提醒: 如果情感再降 2 周，考虑减仓 1/3。"
```

### 不是做什么

❌ 不是 "把持仓列表打印出来"
❌ 不是 "对每只持仓跑一遍 xueqiu analyze"（方向 1）
❌ 不是 "显示涨跌数字"（longbridge 自己就能做）

### 是什么

> 读持仓 → 读情感 → 读基本面 → 发现**不一致** → 告诉你

### 三层不一致检测

```
┌─ Layer 1: 持仓 vs 情感 ──────────────────────────────────┐
│ 你的股票涨了，但市场讨论在变差 → 需要考虑卖               │
│ 你的股票跌了，但市场讨论在变好 → 可以考虑加仓             │
└───────────────────────────────────────────────────────────┘

┌─ Layer 2: 持仓 vs 公告 ──────────────────────────────────┐
│ 你持有的公司发了新公告，但你还没看到                     │
│ 公告里的新数据跟你买入时的逻辑不一致 → 需要重审          │
└───────────────────────────────────────────────────────────┘

┌─ Layer 3: 情感 vs 基本面 ────────────────────────────────┐
│ 情绪回暖但基本面恶化了 → 假反弹                          │
│ 情绪悲观但基本面改善了 → 可能是机会                       │
└───────────────────────────────────────────────────────────┘
```

## 技术方案

### 数据流

```
longbridge-terminal (持仓+成本)
    │  daily snapshot → JSON
    ▼
方向 7 脚本
    │  读持仓 + 读情绪数据 (方向 2) + 读评分 (方向 4 历史)
    ▼
三层不一致检测
    ▼
Markdown 简报 → 飞书推送
```

### 脚本结构

```python
scripts/portfolio_sense.py

def main():
    portfolio = load_longbridge_portfolio()

    for position in portfolio:
        sentiment = load_sentiment_history(position.symbol, days=30)
        score = load_latest_score(position.symbol)
        alerts = load_xueqiu_monitor_alerts(position.symbol)

        analysis = {
            "layer1": check_price_vs_sentiment(position, sentiment),
            "layer2": check_pending_announcements(position.symbol),
            "layer3": check_sentiment_vs_fundamentals(sentiment, score),
        }

        report.sections.append(analysis)

    render_and_push(report)
```

### 依赖

| 依赖 | 来源 | 状态 |
|------|------|------|
| 持仓+成本 | longbridge-terminal CLI | ✅ 已有 |

| 评分历史 | 方向 4 的每周分析 | ⏳ 方向 4 跑起来后才有 |
| 公告列表 | xueqiu-analyzer 数据 | ✅ 已有 |

## 这个方向跟其他 6 个的关系

```
方向 2 (每天) ── 情感数据
    │
方向 4 (每周) ── 评分数据
    │
方向 6 (持续) ── 历史序列
    │
longbridge ─── 持仓数据
    │
    ▼
方向 7: 在这三条数据上做"不一致检测"
        输出一条结论: "你有 18% 浮盈，但市场已经变了"
```

方向 7 是所有方向中**数据依赖性最强**的一个——它不生产自己的数据，全靠其他方向输出。方向 2 和方向 4 先跑起来，方向 7 才能工作。
